# Playwright Web Scraper

This project is an asynchronous web scraping script built with **Python** and **Playwright**.  
It scrapes paginated business records from a target website and exports the extracted data into a JSON file.

---

## Installation

### 1. Clone the repository 
- bash
git clone <your-repo-url>
cd <your-repo-folder>

### 2. Create and activate a virtual environment (optional but recommended)
python -m venv venv
source venv/bin/activate   # macOS / Linux
venv\Scripts\activate      # Windows

### 3. Install dependencies
pip install playwright
playwright install

### How to Run the Script
python scraper.py
